## 🚀 What NaturalQuery Does

NaturalQuery lets you ask questions in plain English and turn them into SQL queries. It is made for Windows users who want a simple way to work with data without writing SQL by hand.

Use it to:
- Ask questions like “Show sales by month”
- Generate SQL for common databases
- Connect natural language with your data tools
- Save time when you need quick answers from tables

NaturalQuery uses an LLM to understand your question and build a query that fits your request.

## 🧰 When You Need a Match to Your Data

NaturalQuery can help with many SQL tasks, but it works best when the words you use match your schema. If your table uses `customer_name`, ask for customers. If your table uses `order_date`, ask about dates or orders.

That simple link between your question and your data helps the LLM build better SQL
